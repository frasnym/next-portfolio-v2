# FrasNym's Portfolio Site Repo

A portfolio website to introduce myself to the world 👋. Please come to visit: [Here](http://frasnym.tech)

## Features
- ✍ Written in TypeScript
- 💻 Builded with NextJs
- 🚀 Deployed on Vercel
- 🐳 Docker support for development

## How to run

- Without docker
```bash
npm run dev
```

- With docker-compose
```bash
docker-compose up
```
